# Plots Directory Organization

This directory contains publication-quality figures for the Conservation Letters manuscript on MPA effects on kelp forest trophic cascades.

## Directory Structure

```
plots/
├── README.md (this file)
├── archive/              # Outdated/deprecated figures
├── diagnostics/          # Model diagnostic plots (DHARMa, NLS diagnostics)
│
├── Main Text Figures (PDF + PNG at 300 DPI):
│   ├── fig_01_mpa_map.{pdf,png}
│   ├── fig_02_data_processing.{pdf,png}
│   ├── fig_03_mean_effects.{pdf,png}
│   └── fig_04_trophic_scatter.png
│
└── Supplemental Figures:
    ├── fig_s01_forest_plot.{pdf,png}
    ├── fig_s02_all_taxa_timeseries.{pdf,png}
    ├── fig_s03_temporal_dynamics.png
    ├── fig_s04_spacetime_heatmap.png
    ├── fig_s05_statistical_transparency.png
    └── fig_s06_appendix_{taxa}.png (5 files, one per taxa)
```

## Diagnostics

The `diagnostics/` subdirectory contains:

- **DHARMa diagnostic plots**: Residual diagnostics for pBACIPS models
- **NLS diagnostic plots**: Nonlinear regression diagnostics
- **Summary diagnostic figure**: `diagnostic_summary.{pdf,png}`

These are generated during the effect size calculation (script 08) and meta-analysis (script 09) for quality control.

## Figure Generation

All figures are generated by running:

```r
source(here::here("code", "R", "10_figures.R"))
```

Or as part of the full pipeline:

```r
source(here::here("code", "R", "run_all.R"))
```

### Export Formats

- **Main text figures**: Exported as both PDF (vector) and PNG (300 DPI raster)
- **Supplemental figures S1-S2**: Exported as both PDF and PNG
- **Supplemental figures S3-S6**: Exported as PNG only (complex multi-panel figures)

### Figure Dimensions

Figures are sized for Conservation Letters specifications:

- Single column width: 80mm (12 cm)
- Double column width: 170mm (17 cm)
- Specific dimensions defined in `10_figures.R` header

## Color Palette

All figures use the colorblind-safe palette defined in `00b_color_palette.R`:

- Purple urchin: `#A463F2`
- Red urchin: `#E05263`
- Kelp: `#6EAA78`
- Sheephead: `#E88C2B`
- Lobster: `#E05263`

## Naming Conventions

- Main text: `fig_{NN}_{description}.{pdf|png}`
- Supplemental: `fig_s{NN}_{description}.{pdf|png}`
- Use lowercase with underscores (snake_case)
- Two-digit zero-padded figure numbers
