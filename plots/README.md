All figures generated by: `source(here::here("code", "R", "run_all.R"))`

## Diagnostics

The `diagnostics/` subdirectory contains model diagnostic plots (DHARMa residuals, NLS diagnostics) generated during effect size calculation for quality control.

## Export Format

- All figures exported as both PDF (vector) and PNG (600 DPI raster)
- Sized for Conservation Letters: single column = 80mm, double column = 170mm
- Colorblind-safe palette defined in `code/R/00b_color_palette.R`
